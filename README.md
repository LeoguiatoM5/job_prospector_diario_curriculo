# QA Job Prospector

Sistema em Node.js para localizar vagas de QA remotas disponíveis para candidatos no Brasil e enviar currículo automaticamente quando existir um e-mail público de recrutamento.

## Regra de negócio

QA + REMOTO + BRASIL = VAGA VÁLIDA

## Tecnologias

- Node.js
- SQLite
- Axios
- Playwright
- Nodemailer
